# cha1 - git config
-----------------------------
### 1. 最重要的安全和隐私问题
>git config --list 查看所有配置, core 配置 是局部的配置, 往上是 局部的配置
##### 对全局的设置
> git config --global 全局设置
> git config --global user.name ''
> git config --global user.mail ''

对局部的设置
> git config --local 全局设置
> git config --local user.name ''
> git config --local user.mail ''
